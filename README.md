# hashtag_-inspector_bot

Telegram-бот для модерации одной forum-ветки в супергруппе и для быстрого переноса обсуждения постов в `GENERAL`.

## Что делает бот

- Следит за сообщениями в одной заданной ветке `message_thread_id`.
- Если в сообщении нет хештега, удаляет сообщение и отправляет короткое предупреждение.
- Если сообщение с хештегом содержит картинку, бот добавляет кнопку `Обсудить в GENERAL`.
- По нажатию на кнопку копирует исходный пост в `GENERAL` или в отдельную ветку обсуждения.
- Кнопка `Обсудить в GENERAL` живет 24 часа и удаляется по TTL независимо от нажатий.
- После успешного переноса обсуждения бот временно показывает кнопку `Перейти в GENERAL`.
- Админ целевой группы может включать и выключать модерацию командами.
- Админ целевой группы может создавать и публиковать посты в группу через личный чат с ботом командой `/post`.

## Основной сценарий

Бот рассчитан на группу с включенными Topics:

1. Пользователи публикуют сообщения в конкретной ветке.
2. Если хештег есть, сообщение остается.
3. Если это пост с изображением, появляется кнопка `Обсудить в GENERAL`.
4. Если хештега нет, сообщение удаляется.

## Требования

- Python 3.11+ желательно.
- Telegram-бот, созданный через BotFather.
- Супергруппа Telegram.
- Если используются ветки, в группе должны быть включены Topics.
- Бот должен быть администратором целевой группы.

## Права бота

Минимально нужны права:

- удаление сообщений;
- отправка сообщений;
- отправка медиа;
- доступ к темам группы.

Для команды `/post` пользователь должен быть администратором целевой группы.

## Установка

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Запуск:

```bash
python3 main.py
```

## Настройка `.env`

Создай файл `.env` в корне проекта.

Пример:

```env
BOT_TOKEN=123456:telegram_bot_token
TARGET_CHAT_ID=-1001234567890
TARGET_THREAD_ID=12345
DISCUSSION_THREAD_ID=
LOG_LEVEL=INFO
LOG_ALL_MESSAGES=false
IGNORE_ADMINS=true
IGNORE_BOTS=true
```

## Переменные окружения

- `BOT_TOKEN` - токен бота.
- `TARGET_CHAT_ID` - ID целевой супергруппы.
- `TARGET_THREAD_ID` - ID ветки, в которой включена модерация по хештегам.
- `DISCUSSION_THREAD_ID` - куда переносить обсуждение после нажатия на кнопку.
  Оставь пустым, чтобы переносить в `GENERAL`.
- `LOG_LEVEL` - уровень логирования, обычно `INFO`.
- `LOG_ALL_MESSAGES` - подробное логирование всех входящих сообщений.
- `IGNORE_ADMINS` - не модерировать админов.
- `IGNORE_BOTS` - не модерировать сообщения от ботов.

## Как узнать `TARGET_CHAT_ID` и `message_thread_id`

Обычно это делают через логирование:

1. Временно включи `LOG_ALL_MESSAGES=true`.
2. Запусти бота.
3. Отправь тестовое сообщение в нужную группу и нужную ветку.
4. Посмотри в логах значения `chat_id` и `thread_id`.
5. Запиши их в `.env`.

## Команды бота

Команды управления:

- `/on` - включить модерацию.
- `/off` - выключить модерацию.
- `/status` - показать текущий статус модерации.

Эти команды работают:

- в группе для админов этой группы;
- в личке бота для админов целевой группы.

## Команда `/post`

Команда `/post` работает только в личных сообщениях с ботом и только для администраторов целевой группы.

Что можно сделать:

- выбрать ветку публикации;
- отправить текст поста в HTML;
- добавить до 10 фото;
- посмотреть превью;
- опубликовать пост в группу;
- очистить фото или сбросить черновик.

### Как пользоваться `/post`

1. Открой личный чат с ботом.
2. Отправь `/post`.
3. Нажми `Указать ветку` и пришли `message_thread_id`.
4. Если нужен `GENERAL`, отправь `0`.
5. Нажми `Написать текст` и пришли текст.
6. При необходимости нажми `Добавить фото` и отправь изображения отдельными сообщениями.
7. Нажми `Превью` для проверки.
8. Нажми `Опубликовать`.

Поддерживается HTML-разметка, например:

```html
<b>Заголовок</b>

Текст поста

<a href="https://example.com">Ссылка</a>
```

## Логика кнопки `Обсудить в GENERAL`

- Кнопка добавляется только для сообщений с хештегом и с изображением.
- Нажатие копирует исходное сообщение в `GENERAL` или в `DISCUSSION_THREAD_ID`, если он задан.
- Исходная кнопка не удаляется сразу после нажатия.
- Кнопка автоматически удаляется через 24 часа.
- Если исходный пост уже недоступен, бот удаляет невалидную кнопку.

## Что считается изображением

Бот считает сообщением с картинкой:

- обычное фото Telegram;
- документ с MIME-типом `image/*`;
- файл с расширением `.jpg`, `.jpeg`, `.png`, `.webp`.

## Что бот не модерирует

- сообщения вне `TARGET_CHAT_ID`;
- сообщения вне `TARGET_THREAD_ID`;
- сервисные сообщения;
- сообщения ботов, если `IGNORE_BOTS=true`;
- сообщения админов, если `IGNORE_ADMINS=true`;
- команды `/on`, `/off`, `/status`.

## Примечания

- Состояние включено/выключено хранится в памяти процесса. После перезапуска бот стартует с `BOT_ENABLED = True`.
- Черновик `/post` тоже хранится только в памяти.
- Для автоудаления кнопки `Обсудить в GENERAL` используется `job_queue` из `python-telegram-bot`.
